Google Drive & Sheets

Simpan/muat data ke Drive, atau impor langsung dari Google Sheets (baca-saja).

Sinkron otomatis dari Sheet publik
Belum aktif. Tempel link sheet publik lalu nyalakan "Aktifkan".
Isi data di Google Sheets → dashboard menarik baris baru otomatis tiap interval. Sheet wajib dibagikan "Siapa saja dengan link → Pelihat". Centang "Perbarui baris lama" hanya jika Sheet adalah sumber kebenaran — sel kosong di Sheet tidak akan menimpa data lama. Edit status di aplikasi sebaiknya dilakukan bila tidak mencentang opsi itu.
Cara cepat — tempel link (tanpa login)
Sheet harus dibagikan "Siapa saja dengan link → Pelihat" (atau dipublikasikan). Tanpa itu, impor tanpa login tidak bisa — pakai opsi "login" di bawah untuk sheet privat. Baris pertama = header.
Untuk sheet privat — perlu login Google
Memeriksa…
OAuth Client ID (dari Google Cloud Console)
Penyimpanan Drive berupa satu file JSON (bukan database). Untuk data sangat besar, sinkron otomatis dimatikan demi keamanan — gunakan tombol manual. Fitur ini hanya aktif saat dashboard dijalankan di origin/hosting Anda sendiri.
Impor dari Google Sheets (baca-saja)
Hanya membaca — sheet asli Anda tidak diubah. Baris pertama dianggap header (Link, Status, Region, dst. dipetakan otomatis). Sheet harus dapat Anda akses dengan akun Google yang sama.
Cara mendapatkan Client ID
1. Buka console.cloud.google.com → buat / pilih project.
2. APIs & Services → Library → aktifkan Google Drive API dan Google Sheets API.
3. OAuth consent screen → isi & tambahkan email Anda sebagai test user.
4. Credentials → Create Credentials → OAuth client ID → tipe Web application.
5. Di Authorized JavaScript origins tambahkan origin hosting Anda (mis. https://domainanda.com atau http://localhost:8080).
6. Salin Client ID, tempel di atas.
